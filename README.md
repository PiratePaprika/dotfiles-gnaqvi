# dotfiles

- Clone to ~/.dotfiles
- Use `.local/bin/dfy` to dotify existing config files to `~/.dotfiles` folder and vice versa
- Run `themer --set` to generate theme. See `.local/bin/themer`.
- Theme can be modified in `~/.config/base16/config.json`
